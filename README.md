# Maze Warface

Maze Warface is a 2D arcade-style tank battle game played in a random maze.

## Gameplay
- Match starts at 0-0.
- A round is won by destroying the opponent tank.
- After each round win:
- The game displays the round winner message.
- Score updates.
- A new random maze is generated.
- The next round begins after a 3-second countdown.
- First to 3 round wins wins the full match.

## Modes
- 1 Player vs AI
- 2 Players local

## Controls
- Player 1 movement: WASD
- Player 1 shoot: Left Click or F
- Player 2 movement: Arrow Keys
- Player 2 shoot: L

## Visual and UI Features
- Bright, colorful arcade presentation
- Distinct tank colors:
- Player 1: Blue
- Player 2: Red
- Smooth movement with fixed-step 30 FPS game loop
- Explosion effect on tank destruction
- HUD at top with score and round number
- Countdown/status display between rounds
- Final victory overlay and Play Again flow
