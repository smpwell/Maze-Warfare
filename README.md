# Maze-Warfare

Maze Warfare is a single-page, browser-based top-down arcade tank shooter.

## Features
- Randomly generated maze each level
- WASD movement and mouse aiming
- Enemy tanks with patrol + attack behavior
- Bullet ricochet mechanics
- Health, score, and level progression
- Power-ups: rapid fire, shield, speed boost
- Retro-style visuals
- Start, game-over, and victory screens
